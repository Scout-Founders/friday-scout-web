# Friday Scout

Options trading intelligence platform powered by a 4-gate analysis system.

Scans 304+ stocks for high-conviction CALL and PUT opportunities using:
- Gate 1: Valuation Dislocation (DCF, Piotroski, Z-Score, RSI)
- Gate 2: Smart Money Confirmation (Analyst grades, insider trading)
- Gate 3: Catalyst Detection (Earnings, news)
- Gate 4: Red Flag Screening (Insider selling, momentum exhaustion)
